Recommended courses

  • NPRG041 Programming in C++ - course of the C++ programming language, expects basic knowledge of programming but no knowledge of C/C++
  • NSWI015 Unix/Linux Programming in C - a harder more advanced course of programming in C
  • NTIN066 Data Structures I - a hard obligatory course that you will need to pass in the summer semester; it has assignments that should be completed in C/C++, so you must be prepared for that by the start of the summer semtester!
