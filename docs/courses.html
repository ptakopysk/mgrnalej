Recommended courses

  • NPRG030 Programming 1 - introductory programming course, does not expect any knowledge of programming; taught in Python.
    The Programming I course is very tightly coupled with the Introduction to Algorithms course. It is expected that you take both of the courses together; it is very hard or impossible to take them separately.
  • NPRG062 Introduction to Algorithms - introductory course of data structures, algorithms and programming techniques. You need to know the stuff taught in this course to be able to pass the obligatory Data Structures I course -- either you know it already, or please go to this course (this is for your own good).
    The Introduction to Algorithms course is very tightly coupled with the Programming I course. It is expected that you take both of the courses together; it is very hard or impossible to take them separately.
  • NPRG041 Programming in C++ - introductory course of programming in C++, expects basic knowledge of programming but no knowledge of C/C++
  • NPRG066 The C Programming Language - introductory course of programming in C, expects basic knowledge of programming but no knowledge of C/C++; taught in summer semester
  • NSWI015 Unix/Linux Programming in C - a more advanced course of programming in C for Unix
  • NSWI120 Principles of Computers - a course delving deep into the architectures of computers, so that you really understand how the processor works, how the programs access RAM, etc; expects no prior knowledge of programming or computers
  • NTIN060 Algorithms and Data Structures 1 - crucial algorithms and data structures: assymptotic complexity (big O notation), trees (BST, AVL, RBT), hashing, sorting, graphs (BFS, DFS), MST, dynamic programming. The Data Structures course will expect that you know especially the basic trees and hashing, and assymptotic complexity. Taught in summer semester.
  • NTIN066 Data Structures I - a hard obligatory course that you will need to pass in the summer semester; it has assignments that should be completed in C/C++, so you must be prepared for that by the start of the summer semtester!
    See also the 2018/2019 website of the course.
